# ✈️ VAAYU Air Tours

**Experience India From the Sky**

VAAYU Air Tours is a futuristic adventure tourism platform that enables users to book luxury aerial rides using electric air taxis (drone-based mobility). This project represents an early-stage MVP for next-generation urban and tourism air transport.

---

## 🚀 Vision

To revolutionize travel by introducing **air taxi-based adventure tourism** in India, combining:

* Luxury experience
* Cutting-edge UAV technology
* Sustainable electric aviation

---

## 🌍 Features

* 🛫 Air Taxi Booking Interface
* 🏔️ Multiple Premium Destinations (Manali, Ladakh, Goa, Udaipur, Himachal)
* 🧭 Interactive UI with smooth animations
* 🛡️ Safety & Aircraft Details Section
* 📅 Booking Modal with form inputs
* 🎥 Aesthetic, modern landing page design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Advanced animations & gradients)
* JavaScript (UI interactions)
* Tailwind CSS (utility styling)
